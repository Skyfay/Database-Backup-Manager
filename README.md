### What is DBackup?

DBackup is a comprehensive, self-hosted backup solution designed to automate and secure your database backups. It provides enterprise-grade encryption (AES-256-GCM), flexible storage options, and intelligent retention policies to ensure your data is always protected and recoverable.

Whether you're running a single MySQL database or managing multiple PostgreSQL, MongoDB, and SQL Server instances, DBackup offers a unified interface with real-time monitoring, granular access control, and seamless restore capabilities.


> **Note**: This project is primarily developed on GitLab. This GitHub repository is a mirror.
> **Main Repository**: [https://gitlab.com/Skyfay/dbackup](https://gitlab.com/Skyfay/dbackup)

## ‚ú® Features

- **Multi-Database Support** ‚Äî MySQL, PostgreSQL, MongoDB, Redis, SQLite, Microsoft SQL Server
- **Enterprise-Grade Security** ‚Äî AES-256-GCM encryption with Encryption Vault, key rotation, and offline recovery kits
- **Flexible Storage** ‚Äî 13+ built-in adapters: S3, Google Drive, Dropbox, OneDrive, SFTP, Rsync, WebDAV, SMB, FTP, and more
- **Smart Retention** ‚Äî Grandfather-Father-Son (GVS) rotation with configurable daily, weekly, and monthly policies
- **API & Webhooks** ‚Äî Trigger backups via REST API with fine-grained API keys ‚Äî ready-made cURL, Bash, and Ansible examples for CI/CD integration
- **Dashboard & Analytics** ‚Äî Interactive charts, real-time progress tracking, storage usage history, and auto-refreshing activity feeds
- **One-Click Restore** ‚Äî Browse backups in the Storage Explorer, verify checksums, and restore to any target database
- **SSO & RBAC** ‚Äî OpenID Connect support (Authentik, PocketID, Generic), user groups, and granular permission system
- **Notifications** ‚Äî Discord, Slack, Microsoft Teams, Generic Webhook, and Email (SMTP) alerts for backups, restores, logins, and system events
- **Configurable Rate Limits** ‚Äî Per-category rate limiting (Auth, API Read, API Write) adjustable from the Settings UI
- **Docker Ready** ‚Äî Multi-arch images (AMD64/ARM64), built-in health checks, and graceful shutdown with backup-safe SIGTERM handling

## üöÄ Quick Start

**Supported Platforms**: AMD64 (x86_64) ‚Ä¢ ARM64 (aarch64)

```yaml
# docker-compose.yml
services:
  dbackup:
    image: skyfay/dbackup:beta
    container_name: dbackup
    restart: always
    ports:
      - "3000:3000"
    environment:
      - ENCRYPTION_KEY=       # openssl rand -hex 32
      - BETTER_AUTH_URL=http://localhost:3000
      - BETTER_AUTH_SECRET=   # openssl rand -base64 32
    volumes:
      - ./backups:/backups
      - ./db:/app/db
      - ./storage:/app/storage
```

```bash
docker-compose up -d
```

Open [http://localhost:3000](http://localhost:3000) and create your admin account.
